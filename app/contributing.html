Contributing

For the Clark Kent type

If you are familiar with Github Pages, the Yeoman authoring environment, and the Jekyll blogging platform, then you will love this!

Creating a new bass tab for OS Bass Tabs is as simple as:

$ grunt post --name "The Doors - Break On Through (To The Other Side)

Of course there is bit more to it: You have to be able to set up the Github repo on your machine, which is all detailed in the project's README.md file.

If you are a developer, and you would like to contribute to the code, or the design, please feel free to make pull requests!

If you find bugs, or any issues with the site, that you would like to see fixed, please refer to the Github issues.

For the Superman type

I understand that most bassists aren't that familiar with coding - and that's completely normal :).

I want to keep this project as lean as possible, so at this stage I am not planning to build an online authoring tool for users - simply because it would take way too long compared to the gains. For that particular reason the website is completely static (meaning there is no server side magic involved) at the moment.

If you would like to put your tabs to the site, the quickest way is to send me an email with your tabs, and I will make sure they will be added (giving you credit of course) to the site as soon as possible.

Minimum requirements of a badass bass tab

There are a few things you have add to your tabs to make sure they really kick ass - here is a quick checklist for you:


artist:       "The artist/band name of the song"
song:         "The title of the song"
album:        "The name of the album"
year:         "The year the song was released"
author:       "Your name"
author_email: "Your email address" (optional)
author_site:  "Your website" (optional)
tuning:       "GDAE"
youtube:      "Full URL of the Youtube video for the song"

Once you have all the above details and your killer bass tabs, you're good to go! You can have a quick look at existing tabs to see how these details are used to build up a superhero bass guitar tab on this site.

A bit about copyright

I would love this project to be a real community driven one, so whatever we do is all for the comminity, and done by the community. This includes bassists wanting to learn and share their tabs, just as well as the developers who want to participate.

The website's source code is made completely open source, and it is released under the MIT licence, which means anyone can use/reuse it for other personal or commercial projects. This includes the tabs created by the comunity.

Please make sure you agree to this before contributing to the site.
