About

So what is OS Bass Tabs?

OS Bass Tabs is a free, open source archive for bass tabs.

A friend of mine showed me his bass guitar a few months ago, and I fell in love with it. Not long after (Christmas 2014) my dear ex surprised me with a beautiful ESP B10 electric bass guitar, and I started practicing like a mad man.

I first used online bass tab archives such as UltimateGuitar.com, but at the same time I couldn't find some of my favourite songs online (and also there is too many annoying ads and way too long load times on those sites), so I started writing my own version of those tabs - using Markdown files of course (I am a geek at the end of the day.)

I am very passionate about web development, and I wanted to see how quickly those markdown files can be turned into an easy to navigate static website, so I started working on OS Bass Tabs.

OS Bass Tabs is all about `open source`. The code for the website is using the latest open source technologies you can get for building static HTML sites: Yeoman, Grunt, Bower and Jekyll (sorry if I am being far too technical here).

The source code of the website can be found on Github, it is also hosted on Github in the form of Github Pages.

Who is OS Bass Tabs for?

This site started off as my primary bass practice tool. I started collecting the bass tabs of my favourite songs, but it is also for everyone, who want to learn those songs.

The website was made open source, so that any geeky bassist can contribute to the list of songs, make improvements and add new features as they wish :)

Why is this website better than any other bass tabs sites?

It isn't better. It is just different:

  • OS Bass Tabs is all about the user and the content. THERE IS NO ADS! I wanted a plaform for myself, which purely focuses on the enjoyment of playing the bass.
  • The development's main focus has always been on the best UX you can get as a user, so there is no distraction (ads, bloating social sharing bollocks). It's just you and the real stuff.
  • Anyone with a bit of front-end knowledge can contribute to the code over Github.
  • New tabs can be added without registration - you need to be a Github geek, or email me and ask nicely to put your tabs on the site :)
