Welcome to OS Bass Tabs!

This is a website I put together primarily to make my bass practicing easier, but I guess it can be helpful for other beginner bassists too.

The concept is fairly simple

You have the tabs accompanied by the songs, so that you not only can see the tabs whilst playing them, but also can listen to the song thanks to lovely people putting them on Youtube :)

You can read more about the project in the About section, and I also welcome everyone to Contribute to this project - whether you want to add your tabs, or to improve the website.
