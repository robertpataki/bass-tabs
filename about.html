OS Bass Tabs

About

So what is OS Bass Tabs?

The concept is fairly simple: OS Bass Tabs is a free, open source archive for bass guitar tabs.

Unlike on any other bass tab website, each bass tab is here is accompanied by the song. The songs here come in the form of embedded Youtube videos: some of them features the original song itself, while others feature brilliant bass covers!

Who is OS Bass Tabs for?

I started this project to better support my bass guitar practicing, but I've realised that it not only can help me, but probably others beginner bassists too.

The website was made open source, so that any geeky bassists can contribute to the list of songs, make improvements to the website, and also add new features if they feel like.

Why is this website better than any other bass tabs sites?

The key differences are:

  • The tab is created and maintained by the open source community
  • Each tab is accompanied by the audio song, so that you can read the tabs and listen to the song itself while practicing
  • There are no ads, no social media sharing bloat - no distraction, only you and the tabs
  • Users of this site can leave comments on each tab page - making this place feel like there is a community behind it - and hopefully that will help us maintain a high quality of bass guitar tabs
  • OS Bass Tabs is mobile and tablet friendly, you can practice on the go :)
